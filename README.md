# FIBOS-EOS-NODES

	通过备份快速启动 EOS 节点

	默认数据，配置存储位置 /blockData/data



## 启动说明

说明:
	
	fibos bp.js   //快去启动一个bp节点

	fibos seed.js // 快速启动一个同步节点

## 工具
	
	FIBOS快速安装

		curl -s https://fibos.io/download/installer.sh |sh
